====================================
    固始县外卖骑手端 - 项目说明
====================================

【项目简介】
这是一个给外卖骑手用的手机APP，用来接单、配送、导航。

【技术类型】
- UniApp 框架（一套代码生成安卓APP）
- Vue3 语法
- 腾讯地图导航

====================================
           📁 文件夹说明
====================================

📁 api/                    → 后端接口（连服务器的代码）
   ├── auth.js             → 登录相关接口
   ├── order.js            → 订单相关接口
   ├── errand.js           → 跑腿订单接口
   ├── user.js             → 用户信息接口
   └── ...

📁 pages/                  → APP的所有页面
   ├── index/              → 首页（骑手工作台）
   ├── login/              → 登录注册页
   ├── orders/             → 外卖订单页
   ├── errands/            → 跑腿订单页
   ├── map/                → 地图导航页
   ├── profile/            → 个人中心
   └── ...

📁 components/             → 公共组件（复用的弹窗等）

📁 utils/                  → 工具函数
   ├── request.js          → 网络请求封装
   ├── storage.js          → 本地存储（存登录信息等）
   ├── socket.js           → 实时消息连接
   ├── rider-auth.js       → 骑手身份判断
   └── ...

📁 static/                 → 静态资源（图标、图片）

📁 config/                 → 配置文件（服务器地址等）

📁 android-all-in-one-demo/ → 安卓原生导航插件源码

📁 unpackage/              → 打包生成的文件（不用管）

====================================
           📄 核心文件说明
====================================

manifest.json              → APP配置（名字、版本、权限）
pages.json                 → 页面路由（所有页面清单）
App.vue                    → APP入口（全局样式、生命周期）
main.js                    → 程序入口
package.json               → 依赖包配置

====================================
           🔧 常用命令
====================================

用 HBuilderX 打开项目，点击：
- 运行 → 运行到手机或模拟器 → 制作自定义基座
- 发行 → 原生APP-云打包

====================================
           ⚠️ 注意事项
====================================

1. 修改代码后保存，APP会自动刷新
2. 真机调试需要连接数据线
3. 打包需要 DCloud 账号和证书
4. 地图导航功能需要腾讯地图SDK
